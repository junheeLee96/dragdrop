# 개요

드래그 앤 드랍 드래그 앤 드랍!!
말로만 들었던 그 기능... 단순히 말로만 들었지 직접 구현해보지는 않았다.

구글에 검색하면 많이들 나오는 기능이지만 직접 나 혼자 생각해 만들어 볼 예정이다.
